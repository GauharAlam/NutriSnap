# Testing Notes

The repo currently uses Node's built-in test runner for backend logic tests.

Run all backend tests:

```bash
npm run test
```

Run the full local verification suite:

```bash
npm run verify
```

Current coverage focus:

- meal nutrition estimation
- progress summary helpers

Recommended next additions:

- auth integration tests
- meals API tests
- goals API tests
- assistant fallback behavior tests
- frontend component and form tests
- end-to-end flows for login, image upload, and dashboard progress
