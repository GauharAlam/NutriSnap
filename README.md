# Fitness and Nutrition Tracking Web App

Monorepo starter for a full-stack fitness and nutrition tracking platform with:

- `client`: React + Vite + Tailwind CSS frontend
- `server`: Node.js + Express.js REST API
- `shared`: shared constants and schemas for cross-layer reuse

## Planned product modules

- JWT authentication
- Food image upload and AI analysis
- Nutrient estimation and meal logging
- Daily dashboard and weekly progress
- Fitness goals and AI diet assistant

## First implementation milestone

- Monorepo scaffolding
- Frontend shell and routes
- Backend API foundation with versioned modules
- Ready for authentication implementation
